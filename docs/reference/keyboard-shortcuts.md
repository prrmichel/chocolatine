# Keyboard Shortcuts

> Productivity shortcuts available in Chocolatine.

## Overview

Chocolatine provides keyboard shortcuts for frequently used actions. These work globally within the app unless noted otherwise.

## Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Ctrl+Enter` | Send message | Ask tab textarea |
| `Ctrl+Tab` | Switch to next chat context | Ask tab |
| `Ctrl+Shift+Tab` | Switch to previous chat context | Ask tab |
| `Ctrl+S` | Save the current prompt | Prompt Library editor |
