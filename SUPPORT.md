# Support

## Support posture

Chocolatine is maintained on a **best-effort** basis with no SLA or guaranteed response time.

## Where to ask for help

- **Bug reports:** open a GitHub issue with reproduction steps.
- **Documentation issues:** open a GitHub issue and include the page path.
- **Feature requests:** welcome as GitHub issues, but treated as non-committal backlog input.

## Before opening an issue

- Confirm you are on a supported platform (Windows 10/11).
- Confirm your setup matches the docs in [README.md](README.md) and [docs/README.md](docs/README.md).
- Check [Troubleshooting](docs/reference/troubleshooting.md) for common fixes.

## Security issues

Do **not** open public issues for vulnerabilities. Follow the private disclosure instructions in [SECURITY.md](SECURITY.md).
