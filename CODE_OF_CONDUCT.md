# Code of Conduct

## Our Commitment

We want Chocolatine to be a welcoming, respectful project for everyone. By participating in this repository, you agree to engage constructively and professionally.

## Expected Behavior

- Be respectful and considerate in issues, discussions, and pull requests.
- Focus on technical topics and project goals.
- Accept feedback gracefully and provide feedback in good faith.
- Assume positive intent and ask for clarification before escalating.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or personal attacks.
- Threats, intimidation, or sustained hostile behavior.
- Publishing private or sensitive information without consent.
- Disruptive trolling, spam, or intentionally derailing discussions.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and review comments.

## Enforcement

Project maintainers may remove, edit, or reject content that violates this Code of Conduct and may limit participation for repeated or severe violations.

For reporting misconduct, use the private contact path in [SECURITY.md](SECURITY.md) and clearly indicate the report is about conduct/moderation.
